# ☔️ 우산 대여 서비스 프론트엔드 프로토타입

React와 `react-router-dom`을 사용하여 구현된 우산 대여 서비스의 프론트엔드 프로토타입입니다. 백엔드 없이 프론트엔드 로직만으로 사용자 흐름을 시뮬레이션합니다.

## 🚀 시작하기

이 프로젝트를 로컬 환경에서 실행하려면 다음 단계를 따르세요.

1.  **저장소 클론:**
    ```bash
    git clone [저장소 URL]
    cd Umbrella/Code/umbrella
    ```
    (참고: `[저장소 URL]`은 실제 저장소 URL로 대체해야 합니다.)

2.  **의존성 설치:**
    프로젝트의 모든 의존성을 설치합니다.
    ```bash
    npm install
    ```

3.  **애플리케이션 실행:**
    개발 모드로 애플리케이션을 실행합니다.
    ```bash
    npm start
    ```
    애플리케이션은 일반적으로 `http://localhost:3000`에서 실행됩니다.

## ✨ 구현된 기능

현재 프로토타입에는 다음과 같은 주요 기능들이 구현되어 있습니다:

*   **공통 레이아웃**: 모든 페이지에 서비스 로고, 타이틀을 포함하는 헤더와 홈/뒤로가기 버튼을 포함하는 푸터가 적용됩니다. (홈 페이지에서는 '뒤로가기' 버튼 비활성화)
*   **동적 라우팅 및 UI**: `react-router-dom`을 사용하여 페이지 간 이동을 처리하며, `state`를 통해 페이지 간 데이터를 전달하여 동적인 UI(예: 페이지 제목)를 구현합니다.
*   **사용자 흐름**:
    *   **대여**: 사용자 정보 등록 -> 우산 선택 -> 대여 정보 확인 -> 대여 완료
    *   **반납**: 사용자 정보 입력 (로그인) -> **'반납 요청' 타이틀** 및 대여 정보 확인 -> 반납 완료
    *   **분실 신고**: 사용자 정보 입력 (로그인) -> **'분실 신고' 타이틀** 및 대여 정보 확인 -> 분실 신고 완료
    *   **고장 신고**: **'고장 신고' 타이틀** 표시 -> 우산 번호 입력 및 **체크박스로 고장 사유 선택** -> 고장 신고 완료
*   **`ThankYouPage` 개선**: 작업 완료 후 동적 감사 메시지를 표시하며, 5초 카운트다운 후 자동으로 홈으로 이동하거나 화면 클릭 시 즉시 홈으로 이동합니다.
*   **더미 데이터**: 백엔드 없이 프론트엔드에서 더미 데이터를 사용하여 흐름을 시뮬레이션합니다.

## 📂 프로젝트 구조

주요 컴포넌트 및 파일 위치:

*   `src/Layout.jsx`: 공통 레이아웃 컴포넌트
*   `src/Layout.css`: 공통 레이아웃 스타일
*   `src/App.jsx`: 메인 라우터 설정
*   `src/User/pages/`: 사용자 관련 페이지 컴포넌트들 (HomePage, UserRegisterPage, UmbrellaSelectPage, RentalConfirmPage, ReportPage, ThankYouPage)
